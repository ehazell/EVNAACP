== Description ==

The Business WP is a creative multipurpose business theme ideal for Portfolio, Business, Corporate, WooCommerce shop, Travel and Creative agencies. The Flexibility and Theme Options such as Services, Woocommerce integration, Contact, Callout, Social, Team, Skills, Testimonials, Slider, Header & footer Customization, typography features provided in theme will amaze you. It’s built the way after customization, that you feel the theme your own. Looking for a Multi Purpose theme? Look no further! See the demo, The only theme you will ever need: https://wordpress.org/themes/the-business-wp/ For more information about the Business WP Pro theme please go to https://www.ceylonthemes.com/product/business-wp-pro

== Installation ==

 1.  In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
 2.  Type in the-business-wp in the search form and press the 'Enter' key on your keyboard.
 3.  Click on the 'Activate' button to use your new theme right away.

==  How to Setup ? == 

  (Fresh installations, Go to Appearance > Customize > Publish to save the starter content such as menu bars, widgets ect:-)

 1.  First, setup your home page by creating a new page from home-page template. Then create another 4 pages from existing About, Contact, Services, Blog templates.

 2.  Then, Select a template as home-page. Go to Appearance > Customize > Home page settings -> Select, Static page and choose name of the home page you have just created above. 

 3.  Create Menus :- Non fresh installations, Go to, Appearance > Menu and Click View all locations. Theme has 2 menu areas called Top and Footer. If a menu already exists, create a menu and assign to top menu. Assign top menu, the pages you have created above except the home-page. Add home page as a link (https://yourdomain.com) in menus. Create a menu and assign to bottom menu. Click save button.

 4.  Each template content can be modified by selecting template by, Customizer -> Manage Pages -> View each page options
 

 * You can edit and manage each featured item by using Customizer -> Theme Options panel. Also, you can manage each page template content from, Customizer -> Manag Pages -> Selecting templates, that each page need to display.

 
 1. Create a Slider, Use existing posts or create 3 posts with title, text and featured image. Go to, Appearance > Customize > Theme Options > Slider. Select created post from post dropdown list. 
 
 2. Create Testimonials, Use existing posts or create posts with title, text and client featured image. Go to, Appearance > Customize > Theme Options > Testimonial. Select created post need to assign to the testimonials.

 3. Add your contact details. Go to, Appearance > Customize > Theme Options > Contact. Details entered here will be displayed in top header and Contact section. Contact forms are not integrated and Use Contact form 7 plugin shortcode to add contact form as you wish.

 4. Add a Hero content, create a page using a page builder such as Site Orign, Elementor or Existing editor. Go to, Appearance > Customize > Theme Options > Hero. Select the created page. Go to, Appearance > Customize > Theme Options > Manage Pages, Select hero content in relevant page.

 5. Add Services, Create posts with title, text and featured image (service image). Go to, Appearance > Customize > Theme Options > Service. Select created posts from post dropdown.

 6. Create Team content, create posts with title, text and  featured image (staff). Go to, Appearance > Customize > Theme Options > Team. Select created posts from post dropdown.

 7. Add Woocommerce products, Go to, Appearance > Customize > Theme Options > Woocommerce Shop. Select product type to show from existing woocommerce products. Add woocommerce widgets, Go to, Appearance -> Widgets -> Drag widgets to Woocommerce Sidebar

 8. Add News/Blog to home page, Go to, Appearance > Customize > Theme Options > News. Select a post category to display latest news / events.
 

==  More customizations: == 

 1.  Go to, to https://www.ceylonthemes.com/wp-tutorials for more guides on how to customize the theme.
 2.  Navigate to Appearance > Customize in your admin panel and customize to taste.
 

== Frequently Asked Questions ==

= Installed The Business WP theme, What is next ? =

 (Fresh installations, Go to Appearance > Customize > Publish to save the starter content such as menu bars, widgets ect:)

 First, setup your home page by creating a new page from home-page template. Then create another 4 pages from existing About, Contact, Services, Blog templates. To select a template as home-page. 

 Go to Appearance > Customize > Home page settings -> Select, Static page and choose name of the home page you have just created above. 

 Go Appearance > Menu, Add about, service, contact and blog pages already created. Do not drag home page to main menu. Add home page as a link (http://yourdomain/) in menus. 

 Then Go to, Appearance -> Customizer. Customize each page by clicking each top menu links and navigation through customized parcial buttons (buttons with blue circle).

= How to manage Home, About, Contact, Service template contents ? =
  Go to, Customizer -> Manag Pages -> Selecting templates, that each page need to display.

= How to create a Slider ? =
 Use existing posts or create 3 posts with title, text and featured image. Go Appearance > Customize > Theme Options > Slider. Select created post from post dropdown list.

= How to enable Home page header image ? =
 Go to, Customizer -> Header image -> Uncheck Disable home header image.

= How to create Testimonials ? =
 Use existing posts or create posts with title, text and client featured image. Go to,Appearance > Customize > Theme Options > Testimonial. Select created post need to assign to the testimonials.

= How to create Services ? =
 Create posts with title, text and featured image (service image). Go to, Appearance > Customize > Theme Options > Service. Select created posts from post dropdown.

= How to add Staff content ? =
 Create posts with title, text and  featured image (staff). Go to, Appearance > Customize > Theme Options > Team. Select created posts from post dropdown.

= Featured Slider image not showing ? =
 Add featured image to each post
 
= Woocommerce cart not showing ? =
 Install woocommerce plugin

= Header image appears with Home slider ? =
 Go to, Appearance > Customize > Header Image > Check, Disable home header image.
